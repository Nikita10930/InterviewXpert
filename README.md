# Interview Expert

## 📌 Description
Interview Expert is a web application that helps students prepare for interviews by providing questions and answers.

## 🚀 Features
- View interview questions
- Show answers
- Simple and clean UI

## 🛠 Technologies Used
- HTML, CSS, JavaScript
- Python (Flask)
- CSV Dataset


### Step 1:
Clone the repository
git clone <your-repo-link>

### Step 2:
Go to project folder
cd Interview-Expert

### Step 3:
Run backend
python app.py

### Step 4:
Open frontend
Open index.html in browser
